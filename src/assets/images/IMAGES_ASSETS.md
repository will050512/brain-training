## 📋 圖片規格要求

### 檔案格式

| 格式 | 使用場景 | 說明 |
|------|----------|------|
| `.svg` | **優先** | 向量圖形，可縮放不失真，適合圖示和 UI 元素 |
| `.png` | **次選** | 需要透明背景的點陣圖 |
| `.webp` | **推薦** | 照片類圖片，檔案小品質好 |
| `.jpg` | **備選** | 照片類圖片，不需透明背景時使用 |

### 尺寸規格

| 用途 | 建議尺寸 | 說明 |
|------|----------|------|
| 遊戲圖示 | 64×64 px ~ 128×128 px | 清晰可辨識 |
| 配對卡片 | 120×160 px | 4:5 比例 |
| 遊戲背景 | 1920×1080 px | 16:9 全螢幕 |
| UI 按鈕 | 48×48 px ~ 64×64 px | 觸控友善 |
| 角色/物件 | 依遊戲需求 | 保持一致性 |

### 技術規格

| 參數 | 建議值 | 說明 |
|------|--------|------|
| 色彩模式 | sRGB | 標準網頁色彩空間 |
| PNG 壓縮 | 8-bit | 減少檔案大小 |
| WEBP 品質 | 80-90% | 平衡品質與大小 |
| 最大檔案 | < 200 KB | 單一圖片檔案 |

## 🎨 設計風格指南

### 整體風格
- **友善可愛**：適合長者使用，避免過於複雜或陰暗
- **高對比度**：確保視力較弱者也能看清
- **圓角設計**：避免尖銳邊角，給人安全感
- **色彩鮮明**：使用明亮但不刺眼的顏色

### 配色建議

| 顏色類型 | 建議色值 | 用途 |
|----------|----------|------|
| 主色調 | `#4F46E5` (靛藍) | 主要按鈕、標題 |
| 成功色 | `#10B981` (翠綠) | 正確回饋 |
| 錯誤色 | `#EF4444` (紅色) | 錯誤提示 |
| 警告色 | `#F59E0B` (橙色) | 警告提示 |
| 背景色 | `#F3F4F6` (淺灰) | 頁面背景 |
| 文字色 | `#1F2937` (深灰) | 主要文字 |

### 無障礙設計
- **文字與背景對比度** ≥ 4.5:1 (AA 級)
- **大型文字對比度** ≥ 3:1
- **觸控目標** ≥ 44×44 px
- **圖示需搭配文字說明**

## 🔧 圖片優化工具

### SVG 優化
```bash
# 使用 svgo 優化 SVG
npx svgo input.svg -o output.svg

# 批次優化資料夾
npx svgo -f ./input-folder -o ./output-folder
```

### PNG/WEBP 優化
```bash
# 使用 sharp 轉換 PNG 到 WEBP
npx sharp-cli --input "*.png" --output webp --format webp --quality 85

# 使用 imagemin 壓縮 PNG
npx imagemin "*.png" --out-dir=./compressed --plugin=pngquant
```

### 線上工具
- **SVG**: [SVGOMG](https://jakearchibald.github.io/svgomg/)
- **PNG**: [TinyPNG](https://tinypng.com/)
- **WEBP**: [Squoosh](https://squoosh.app/)

## 📁 命名規則

### 檔案命名
- 使用**小寫英文**
- 使用**連字號**分隔單字（kebab-case）
- 格式：`{對象}-{狀態/變體}.{副檔名}`

### 命名範例
```
✅ 正確命名
mole-normal.svg
mole-happy.svg
card-back.png
button-primary-hover.svg

❌ 錯誤命名
Mole.svg           （不要大寫）
mole_normal.svg    （不要底線）
圖片1.png          （不要中文）
buttonHover.svg    （不要駝峰式）
```

## ⚠️ 注意事項

1. **版權問題**：確保所有圖片均為原創或已取得授權
2. **一致性**：同系列圖片風格保持一致
3. **備援方案**：若圖片載入失敗，系統會使用 CSS 繪製的備援圖形
4. **響應式**：SVG 優先，確保在不同螢幕尺寸顯示良好
5. **測試**：在不同裝置和瀏覽器測試顯示效果
