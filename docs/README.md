# Docs Index / 文件索引

本目錄整理專案文件，依用途分類。
This directory organizes project documents by purpose.

## 概覽 / Overview
- `../README.md`：專案總覽與主要功能/路由
  `../README.md`: Project overview and main features/routes.
- `PROJECT_REVIEW.md`：現況盤點、缺失與 UI/UX 改善建議
  `PROJECT_REVIEW.md`: Current-state review, gaps, and UI/UX improvement notes.

## 開發 / Development
- `DEVELOPER_GUIDE.md`：開發環境、流程、常見問題
  `DEVELOPER_GUIDE.md`: Dev setup, workflow, and FAQ.
- `STORYBOOK.md`：Storybook 使用方式
  `STORYBOOK.md`: Storybook usage.
- `pwa-update-fix.md`：PWA 更新流程修正說明
  `pwa-update-fix.md`: PWA update flow fix notes.

## 產品 / UX / 視覺 / Product, UX, Visual
- `DESIGN_SYSTEM.md`：設計系統與元件原則
  `DESIGN_SYSTEM.md`: Design system and component principles.
- `RESPONSIVE_DESIGN_SYSTEM.md`：響應式與觸控設計規範
  `RESPONSIVE_DESIGN_SYSTEM.md`: Responsive and touch design guidelines.
- `../ASSET_GUIDE.md`：素材實際使用清單（以程式為準）
  `../ASSET_GUIDE.md`: Asset usage list (source of truth is code).
- `ASSET_REQUIREMENTS.md`：素材規格與目前使用狀態
  `ASSET_REQUIREMENTS.md`: Asset specs and current usage status.
- `ASSET_UI_USAGE.md`：素材優先順序與 UI 使用位置
  `ASSET_UI_USAGE.md`: Asset priority and UI placement.
- `SCORING.md`：統一評分與權重
  `SCORING.md`: Scoring normalization and weights.

## 資料 / 整合 / Data & Integration
- `data-schema.md`：資料結構與 Google Sheet 欄位
  `data-schema.md`: Data schema and Google Sheet fields.
- `app-embedding.md`：App/WebView 內嵌與外部登入
  `app-embedding.md`: App/WebView embedding and external login bridge.
- Apps Script 範例：本 repo 未提供 Apps Script 檔案，請自行建立 Web App；欄位與 payload 請參考 `data-schema.md` 與 `app-embedding.md`。
  Apps Script example: This repo does not include an Apps Script file. Create your own Web App and follow field/payload specs in `data-schema.md` and `app-embedding.md`.

  `src/assets/audio/games/README.md`: Audio list and notes.
- `src/assets/images/IMAGES_ASSETS.md`：圖片素材說明
  `src/assets/images/IMAGES_ASSETS.md`: Image asset notes.
