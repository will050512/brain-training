# Storybook / Storybook

Storybook is used to document and iterate on UI components in isolation.
Storybook 用於獨立記錄與迭代 UI 元件。

## Run Locally / 本地啟動

```bash
npm run storybook
```

## Build Static Docs / 建置靜態文件

```bash
npm run build-storybook
```

## Notes / 備註

- Storybook is configured under `.storybook/`.
- Global styles are loaded from `src/style.css`.
- Stories live in `src/stories/`.

- Storybook 設定在 `.storybook/`。
- 全域樣式載入自 `src/style.css`。
- Stories 放在 `src/stories/`。
